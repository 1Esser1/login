# PriorIT — Frontend

React frontend for the PriorIT IT task prioritization platform.

## Tech Stack
- React 18 + Vite
- Tailwind CSS v4
- Zustand (state management)
- Axios (HTTP client)
- Lucide React (icons)
- Recharts (charts)

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173`

## Pages
- `/login` — Authentication
- `/register` — Account request
- `/dashboard` — KPI overview
- `/tasks` — Task submission
- `/backlog` — Ranked task list
- `/scoring` — AI scoring results
- `/admin` — Admin approval panel
